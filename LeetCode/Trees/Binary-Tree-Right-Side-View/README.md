## 💡 Approach — Breadth-First Search (BFS)

> 🧠 *Generated by UniCode AI*

**Key Insight:** The key insight is to use a queue to perform a level-order traversal of the binary tree, and append the rightmost node's value to the result at each level.

### Algorithm Walkthrough

1. Initialize an empty queue and push the root node into it.
2. While the queue is not empty, process each level of the tree. The size of the queue at each level represents the number of nodes at that level.
3. For each node at the current level, check if it's the rightmost node (i.e., it's at the index equal to the size of the queue minus 1). If so, append its value to the result.
4. Push the left and right children of the current node into the queue, if they exist.

### Complexity Analysis

## ✅ Accepted Solution 

```cpp
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        vector<int> ans;
        if (!root) return ans;
        
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int s = q.size();
            for (int i = 0; i < s; i++) {
                TreeNode* node = q.front();
                q.pop();
                
                if (i == s - 1) {
                    ans.push_back(node->val);
                }
                
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
        }
        return ans;
    }
};
```
