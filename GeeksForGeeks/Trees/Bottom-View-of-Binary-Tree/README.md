## 📋 Problem Statement

You are given the root of a binary tree, and your task is to return its bottom view. The bottom view of a binary tree is the set of nodes visible when the tree is viewed from the bottom.

Note: If there are multiple bottom-most nodes for a horizontal distance from the root, then the latter one in the level order traversal is considered.

Examples :

Input: root = [1, 2, 3, 4, 5, N, 6]
    
Output: [4, 2, 5, 3, 6]
Explanation: The Green nodes represent the bottom view of below binary tree.
    

Input: root = [20, 8, 22, 5, 3, 4, 25, N, N, 10, 14, N, N, 28, N]
    
Output: [5, 10, 4, 28, 25]
Explanation: The Green nodes represent the bottom view of below binary tree.
    

Constraints:
1 ≤ number of nodes ≤ 105
1 ≤ node->data ≤ 105

**Key Insight:** This approach works by using a breadth-first search (BFS) traversal of the binary tree and storing the nodes at each horizontal distance in a map, allowing for efficient retrieval of the bottom view.

### Algorithm Walkthrough

1. Step 1: Initialize a map to store the nodes at each horizontal distance and a queue to perform BFS traversal. The queue is initialized with the root node and its horizontal distance (0).
2. Step 2: Perform BFS traversal by popping nodes from the queue, updating the map with the current node's data and horizontal distance, and pushing the left and right child nodes into the queue with their respective horizontal distances.
3. Step 3: After the BFS traversal is complete, iterate through the map to retrieve the nodes at each horizontal distance, which represents the bottom view of the binary tree.

/*
Definition for Node
class Node {
public:
    int data;
    Node* left;
    Node* right;

    Node(int val) {
        data = val;
        left = right = nullptr;
    }
};
*/

class Solution {
  public:
    vector<int> bottomView(Node *root) {
        // code here
        vector<int> ans;
        if (!root) return ans;
        
        map<int, int> mp;
        queue<pair<Node*, int>> q;
        q.push({root, 0});
        
        while (!q.empty()) {
            Node* node = q.front().first;
            int line = q.front().second;
            q.pop();
            
            
            mp[line] = node->data;
            
            
            if (node->left) q.push({node->left, line - 1});
            if (node->right) q.push({node->right, line + 1});
        }
        
        for (auto it : mp) {
            ans.push_back(it.second);
        }
        
        return ans;
    }
};
